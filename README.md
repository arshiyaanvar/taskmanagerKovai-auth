# TaskManagerKovai - OAuth Version

## Overview
This is an extended version of the TaskManagerKovai application with real Google OAuth authentication.

## Features
- Google Sign-In using OAuth 2.0
- Create tasks
- View tasks
- Update task status (Planned, In Progress, Complete)
- Priority-based task organization

## Tech Stack
- Python
- Streamlit
- Authlib (OAuth)
- JSON (local storage)

## Note
This version demonstrates production-ready authentication.
Due to HTTPS constraints, this version is not deployed.
